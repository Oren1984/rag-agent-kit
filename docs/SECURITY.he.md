# אבטחה

## חוקים ברירת מחדל (Secure-by-default)
- השירות לא יעלה בלי `RAG_API_KEY`
- לכל בקשה מוגנת חייבים `X-API-Key`
- ברירת המחדל בדוקר מאזינה ל־`127.0.0.1` בלבד
- Web Search כבוי כברירת מחדל

## מה לא עושים
- לא חושפים את ה־API לאינטרנט בלי Reverse Proxy + TLS
- לא פותחים CORS ל־`*` בפרודקשן
- לא מריצים debug בפרודקשן
